WinRAR 5.91 beta 1

Simply. Most. Popular. Archiver. In the world. Everyone knows him, everyone used him. There are often reviews on the network, they say, who needs a paid WinRAR when there is a free 7-zip? Convenience guys. 7-zip is certainly good, but WinRAR is a classic. He was and, I am sure, will be archiver No. 1 for a long time to come. This is a vivid example of how good software should be. Naturally, WinRAR supports unpacking all modern archive formats, but it can only pack in zip and its own rar format. It has a simple and intuitive interface that can be changed using hundreds of different themes. I will not go into technical details - they can be found in the official information.





Assembly information
General information:

Type: x86 / x64 installation; unpacking portable by PortableAppZ
Languages: Russian, English, Ukrainian
Activation: CORE

Additionally:

Implemented the pickup of settings from the WinRAR5PREFS folder:

1. WinRAR settings - Settings.reg file
   You can get it by running the program and using
   the Settings> Export parameters to file command;
   
2. WinRAR themes
   Unpack into the WinRAR5PREFS \ Themes folder.
   The structure will look like this:
   WinRAR5PREFS \ Themes \ Theme name \ bmp, ico and other files.
   
3. Design styles and localization - RarLng.dll file.
   For advanced users who change the interface with the resource editor
   
4. License file - RarReg.key
   Generated by keygen or purchased officially in your name or organization

Command line parameters:

S / IEN
Silent installation of the English version: / S / IEN Silent installation of the Ukrainian version: / S / IUA
Unpacking the Russian portable version: / S / PRU
Unpacking the English portable version: / S / PEN
Unpacking the Ukrainian portable version: / S / PUA
Selecting the installation location: / D = WAY

The / D = PATH parameter should be indicated with the most recent
Example: WinRAR.v5.50.exe / S / IR / D = C: \ MyProgram

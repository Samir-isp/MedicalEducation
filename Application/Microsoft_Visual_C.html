Microsoft Visual C ++ 2005-2008-2010-2012-2013-2019 Redistributable Package 06/03/2020

The Microsoft Visual C ++ Redistributable Package installs the Visual C ++ library runtime components necessary to run applications developed using Visual C ++ on a computer without the Visual C ++ environment installed. In short, this is what should be installed with the drivers immediately after Windows.







Assembly information
In this assembly, the latest official 32-bit and 64-bit versions of Microsoft Visual C ++ 2005-2008-2010-2012-2013-2019 Redistributable Package. This assembly also contains False plug-ins from previous versions of the Microsoft Visual C ++ Redistributable Package, so that program installers can easily install without requiring installed systems, previous versions of the Microsoft Visual C ++ Redistributable Package.

Note:
The assembly does not include stubs (False) from previous versions of 2019 (2017,2015) Redistributable Package. The problem is that new versions of 2019 (2017,2015) Redistributable Package do not allow previous versions of 2019 (2017,2015) Redistributable Package to be installed, which as a result gives an error when installing programs that have previous versions of 2019 (2017,2015) in the installer Redistributable Package. There is no workaround for this problem. In order to install a program that has previous versions of 2019 (2017,2015) Redistributable Package in the installer, you need to remove new versions of 2019 (2017,2015) Redistributable Package and only then, install the program. After installing the program, you will again have to install the new version 2019 (2017,2015) Redistributable Package. And so, you have to do with each such program,

The Microsoft Visual C ++ 2019 Redistributable Package replaces the Microsoft Visual C ++ 2015 and 2017 Redistributable Package.

Assembly features:

- Supported operating systems: 32-bit and 64-bit.
- When installing the assembly, the following will be installed: 32-bit versions of Microsoft Visual C ++ 2005-2008-2010-2012-2013-2017 Redistributable Package, on 32-bit operating systems, and 32-bit and 64-bit versions of Microsoft Visual C ++ 2005-2008-2010-2012-2013-2017 Redistributable Package, on 64-bit operating systems.
- During the installation, it searches and removes previously installed 32-bit and 64-bit versions of Microsoft Visual C ++ 2005-2008-2010-2012-2013-2015-2017 Redistributable Package.
- By default, the assembly is installed with the stubs hidden (False).
- Installation of Microsoft Visual C ++ 2012 Redistributable 11.0.50727 False will occur only on operating systems starting with Windows Vista SP 2 and higher.

If Microsoft Visual C ++ installed on your system is not deleted, then use the MicrosoftFixit.ProgramInstallUninstall utility  to remove them.

/HideOriginal - скрытие оригинальных Microsoft Visual C++ Redistributable Package в "Установка и Удаление программ" (Работает только при установке Microsoft Visual C++ Redistributable Package). Ключ не чувствителен к регистру.

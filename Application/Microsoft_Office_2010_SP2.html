General information:

Languages : pro plus - ru / en / ua; standard - ru / en

To speed up the installation, it is recommended to turn off the antivirus

Command-line options:

-S or-S1 - hide the window with the progress of unpacking temporary files
-S2 - show the window with the progress of unpacking temporary files
-S or -S1 or -S2 - required parameter

/ XO86U - silent installation of Office (x86) with updates
/ XO64U - silent installation of Office (x64 ) with updates
/ XO86N - silent installation of Office (x86) without updates
/ XO64N - silent installation of Office (x64) without updates

/ XV86U - silent installation of Visio (x86) with updates
/ XV64U - silent installation of Visio (x64) with updates
/ XV86N - silent installation of Visio (x86) without updates
/ XV64N- silent installation of Visio (x64) without updates

/ XP86U - silent installation of Project (x86) with updates
/ XP64U - silent installation of Project (x64) with updates
/ XP86N - silent installation of Project (x86) without updates
/ XP64N - silent installation of Project (x64 ) without updates

Silent installation is carried out using .msp files and, if necessary, you can configure them using the deployment center. Details of using the OCT can be found at the . / XOMSP = "FULL_path_do_msp" - specify the .msp file for installing Office / XVMSP = "FULL_path_do_msp" - specify the .msp file for installing Visio / XPMSP



= "FULL_path_to_msp" - specify the MSP file for installing Project
Example : Install.exe / XO86U /XOMSP="C:\Excel_PowerPoint_Word.msp "

MSP files already included in the assembly:
AEW.msp - Access, Excel, Word
EPW.msp - Excel , PowerPoint, Word
EW.msp - Excel, Word
Example of use : Install.exe -S / XO86U /XOMSP="AEW.msp "

/ XUP2010 - update of installed Office 2010
/ XTAB - installation of Office Tab
/ XUBRU - installation of the Russian version of UBit Menu
/ XUBEN - installation of the English version of UBit Menu
/ XKMS - activation

Examples :

Install.exe -S / XO86U / XV86N / XTAB / XKMS
Install Office with updates, Visio without updates, Office Tab, activate

Install.exe -S / XV86U / XVMSP = "C: \ my msp \ custom_visio.msp"
Install Visio with your own .msp file

If you use unpacked or ISO assembly version, pass the parameters to the AutorunHelper.exe file , but exclude -S [1] [2]

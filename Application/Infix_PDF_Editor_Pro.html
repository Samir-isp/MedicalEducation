Infix PDF Editor Pro 7.5.1

Infix PDF Editor Pro is a convenient program with very rich functionality, designed to open, edit and save files in PDF format. Among the main features of the program, in addition to the standard set inherent in all text editors, the following can be distinguished: the ability to work with graphics, insert fragments of other PDF documents, merge several documents into one, search and replace text in several documents, censor text, and much more.

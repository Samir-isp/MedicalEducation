MEDISET®

MEDISET®




Mediset est une gamme de sets de soins complets permettant la réalisation de pansement, perfusion et sondage à domicile.
Boîte unitaire contenant :
  • Un set stérile sous blister, constitué d'un ensemble de composants à usage unique, le tout dans un ordre déterminé, dans le but de réaliser un soin ou un geste particulier (par exemple une perfusion, un sondage urinaire, ou un pansement) dans des conditions rigoureuses d'asepsie et de sécurité.
  • Hors blister : un ensemble d'éléments apportant notamment une « solution système » pour simplifier et sécuriser le soin dans le respect de l'environnement (gel hydroalcoolique, sac à déchets de soins [perforants interdits] - conforme à la nouvelle norme AFNOR NF, février 2001).
  • Un ensemble d'informations proposées aux patients et aux soignants (protocole d'utilisation, numéro Vert, aide à la prescription).
Stérilisé à l'oxyde d'éthylène (EO).
Produits médico-techniques destinés à apporter une solution globale à la réalisation d'une perfusion (en infectiologie, oncologie, traitement de la douleur, pneumologie, ORL, cardiologie, hématologie, rhumatologie et autres affections chroniques lourdes), d'un sondage vésical ou d'un pansement technique, à domicile.
Se conformer strictement à la prescription médicale.
Vérifier la date de péremption et l'intégrité de l'emballage avant usage. A jeter conformément à la réglementation (déchets issus d'activités de soins).

Préparer le sac collecteur de déchets.

Suivre les instructions proposées dans le protocole d'utilisation.

Éliminer le matériel souillé dans le sac collecteur (hors coupants, piquants, tranchants).

Mediset se compose de trois gammes, matérialisées par des codes couleurs :

SETS DE REMPLISSAGE

Sets destinés à remplir les appareillages choisis (perfuseurs, diffuseurs portables, systèmes actifs...) du médicament injectable prescrit.
Tarif LPPR :
  • Set de remplissage/reconstitution pour perfuseur ou diffuseur : 8.99 euros TTC (ACL 748086.5).
  • Set de remplissage/reconstitution pour pousse-seringue : 15.24 euros TTC (ACL 748087.1).
  • Set de remplissage/reconstitution pour systèmes actifs hors pousse-seringue : pris en charge avec le forfait « accessoires pour pompes ambulatoires » à concurrence de 35.06 euros TTC (ACL 780822.5).

SETS DE POSE

Sets destinés à poser un cathéter court sur une voie veineuse périphérique ou sous-cutanée, ou une aiguille spécifique sur une voie centrale.
Tarif LPPR :
  • Set de pose sur voie périphérique IV ou SC : 11.43 euros TTC (ACL 748089.4).
  • Set de pose « veines fragiles » sur voie périphérique IV ou SC avec perfuseur 3V intégré : 13.46 euros (ACL 775128.7).
  • Set de pose sur voie périphérique IV ou SC avec perfuseur 3V intégré : 13.46 euros (ACL 763245.3).
  • Set de pose sous-cutanée avec perfuseur 3V intégré : 13.46 euros (ACL 436374.5).
  • Set de pose sur chambre implantable ou cathéter central 8.08 euros (ACL 748090.2).
  • Set de pose sur chambre implantable avec aiguille de Huber type II 20G : 14.84 euros (ACL 748091.9).

SETS D'HÉPARINISATION

Sets destinés à effectuer un rinçage ou une héparinisation (héparine non contenue dans le set) de la voie d'accès utilisée pour la perfusion. Contient une pince de retrait sécurisé Medisecur®.
Tarif LPPR :
  • Set de rinçage/héparinisation : 6.56 euros (ACL 748092.5).
  • Set de rinçage/héparinisation pour chambre implantable avec aiguille de Huber type II 20 G : 13.32 euros (ACL 444251.6).

SETS DE SONDAGE URINAIRE

Sets destinés à effectuer la toilette hygiénique et antiseptique, avant la pose d'une sonde urinaire. La sonde n'est pas incluse dans les sets.
Tarif LPPR :
  • Set de sondage urinaire, sonde non incluse : 6.10 euros (ACL 436375.1).
  • Set de sondage urinaire avec une poche de recueil 2 litres intégrée, sonde non incluse : 8.30 euros (ACL 436376.8).

Mise en garde :
Le remboursement de MediSet Sondage exclut le remboursement des sondes PVC ou polyuréthane avec traitement de surface hydrophile autolubrifiant.
Codes LPPR : 1189287, 1165186, 1130325, 1111836, 1169445, 1140909.

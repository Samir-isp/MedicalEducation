MOLITG-LES-BAINS

MOLITG-LES-BAINS

Eaux sulfurées sodiques, radioactives, riches en plancton thermal, 37,5 °C.
Rhumatologie (RH) :
  • Arthroses, rhumatismes dégénératifs ; articulations périphériques ; rhumatismes inflammatoires (en dehors des poussées évolutives) ; rhumatismes abarticulaires.
  • Autres indications : ostéopathie raréfiante simple ou compliquée de tassements vertébraux, ostéoporose.
Séquelles de traumatismes ostéo-articulaires.
Dermatologie (DER) :
Eczémas secs ou suintants. Affections dermatologiques diverses. Ulcères variqueux, plaies atones, escarres. Séquelles de brûlures. Cicatrices vicieuses, chéloïdiennes. Retard de cicatrisations. Suites de chirurgie esthétique. Lésions chroniques du cuir chevelu.
Voies respiratoires (VR) :
ORL :
  • Rhinopathies chroniques de l'adulte et de l'enfant ; sinusites ; pharyngites ; laryngites ; pathologie auriculaire et tubotympanique.
  • Indications d'actualité : accidents barotraumatiques otologiques et sinusiens ; suites de la nouvelle chirurgie fonctionnelle sinusienne.
Pneumologie : Bronchopathies chroniques : post-tabagiques, séquelles de tuberculose ; bronchopneumopathies chroniques obstructives (BPCO) ; emphysèmes ; asthmes (adulte et enfant) ; dilatations des bronches (DDB) ; pneumoconioses (silicose).
Traitement du double handicap :
Soit Rhumatologie + Voies respiratoires (RH + VR), soit Rhumatologie + Dermatologie (RH + DER), soit Dermatologie + Rhumatologie (DER + RH), soit Voies respiratoires + Rhumatologie (VR + RH).
L'eczéma aigu n'est pas une contre-indication.
Cure de boisson ; bains en cabine individuelle, avec douche sous immersion, avec irrigation vaginale ou anale ; douches générale au jet, locale (manu et pédi-douches), pénétrante générale, locale prélaryngée, pharyngienne, filiforme ; pulvérisation externe générale ; massages sous l'eau ; cataplasmes de boue ; gargarisme ; irrigation nasale ; aérosol sonique ; humage individuel ; pulvérisation nébulisation ; radio-vaporium ; drainage postural ; rééducation respiratoire ; insufflation de trompe ; méthode de déplacement de Proetz ; applications locales d'eau planctonée enrichie ; piscine de mobilisation, douche de forte pression sous immersion. Bain de boue général en apesanteur.
